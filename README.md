# 法拍房销售助手

## 部署说明

### 方式一：Vercel（推荐）

1. 访问 https://vercel.com
2. 用 GitHub 账号登录
3. 点击 "Add New Project"
4. 选择 "Import Git Repository"
5. 上传本文件夹到 GitHub，或直接拖拽文件夹到 Vercel
6. 点击 Deploy
7. 部署完成后会生成链接，如：`https://fapai-assistant.vercel.app`

### 方式二：GitHub Pages

1. 在 GitHub 创建新仓库
2. 上传本文件夹内容
3. 进入 Settings → Pages
4. Source 选择 "Deploy from a branch"
5. Branch 选择 "main"，文件夹选 "/ (root)"
6. 保存后会生成链接

### 方式三：Netlify

1. 访问 https://app.netlify.com
2. 拖拽本文件夹到页面
3. 自动部署，生成链接

## 使用说明

1. 打开部署后的链接
2. 点击右上角 "⚙️ 设置"
3. 填入硅基流动 API Key
4. 开始使用

## 文件说明

- `index.html` - 主程序（纯前端，无需后端）
- `vercel.json` - Vercel 部署配置
- `README.md` - 本文件
